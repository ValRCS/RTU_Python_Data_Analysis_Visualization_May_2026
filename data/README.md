# Data

## titanic.csv

Source: Data Science Dojo public datasets repository.

- Raw CSV: https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv
- Upstream repository: https://github.com/datasciencedojo/datasets
- Downloaded: 2026-05-10
- Shape: 891 rows, 12 columns
- Target column: `Survived`

This is the commonly used Titanic passenger survival dataset for introductory data analysis and machine learning exercises.

## penguins.csv

Source: palmerpenguins R package by Allison Horst, Alison Hill, and Kristen Gorman.

- Raw CSV: https://raw.githubusercontent.com/allisonhorst/palmerpenguins/main/inst/extdata/penguins.csv
- Upstream repository: https://github.com/allisonhorst/palmerpenguins
- Package documentation: https://allisonhorst.github.io/palmerpenguins/
- License: CC0
- Downloaded: 2026-05-10
- Shape: 344 rows, 8 columns
- Class column: `species`

This is the curated Palmer Penguins dataset. It is small, clean, and useful for scatterplots, distributions, group comparisons, correlation, and visual classification intuition.

## gapminder.csv

Source: `gapminder` R package by Jennifer Bryan, using an excerpt from Gapminder data.

- Source TSV: https://raw.githubusercontent.com/jennybc/gapminder/main/inst/extdata/gapminder.tsv
- Upstream repository: https://github.com/jennybc/gapminder
- Package documentation: https://jennybc.github.io/gapminder/
- Data license: Creative Commons Attribution 3.0 Unported, according to the package documentation
- Downloaded: 2026-05-10
- Local format: converted from TSV to CSV
- Shape: 1,704 rows, 6 columns
- Time range: 1952 to 2007, in 5-year steps
- Grouping columns: `country`, `continent`

This is a compact teaching dataset for time series, country comparison, line charts, filtering by region or year, and realistic public data workflows. It is a historical teaching excerpt, not a current source of socioeconomic indicators.
