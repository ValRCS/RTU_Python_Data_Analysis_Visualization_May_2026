# Data Folder

This folder contains local copies of the datasets used in the course workbooks.

For teaching guidance, dataset roles, caveats, and Google Colab raw URLs, see `../docs/DATASETS.md`.

## Files

### `titanic.csv`

- Source: Data Science Dojo public datasets repository
- Raw source: https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv
- Upstream repository: https://github.com/datasciencedojo/datasets
- Downloaded: 2026-05-10
- Shape: 891 rows, 12 columns
- Main teaching use: Day 1 Pandas analysis, filtering, missing values, grouping, and short conclusions

### `penguins.csv`

- Source: Palmer Penguins dataset by Allison Horst, Alison Hill, and Kristen Gorman
- Raw source: https://raw.githubusercontent.com/allisonhorst/palmerpenguins/main/inst/extdata/penguins.csv
- Upstream repository: https://github.com/allisonhorst/palmerpenguins
- Documentation: https://allisonhorst.github.io/palmerpenguins/
- License: CC0
- Downloaded: 2026-05-10
- Shape: 344 rows, 8 columns
- Main teaching use: Day 2 visualization, group comparison, scatterplots, distributions, and visual storytelling

### `gapminder.csv`

- Source: `gapminder` R package by Jennifer Bryan, using an excerpt from Gapminder data
- Source TSV: https://raw.githubusercontent.com/jennybc/gapminder/main/inst/extdata/gapminder.tsv
- Upstream repository: https://github.com/jennybc/gapminder
- Documentation: https://jennybc.github.io/gapminder/
- Data license: Creative Commons Attribution 3.0 Unported, according to the package documentation
- Downloaded: 2026-05-10
- Local format: converted from TSV to CSV
- Shape: 1,704 rows, 6 columns
- Main teaching use: optional bonus notebook for line charts, public-data trends, and visual analysis
